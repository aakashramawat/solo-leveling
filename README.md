## WHAT IS THIS?

A locally hosted, self-improvement system built in the aesthetic of the *Solo Leveling* anime and manhwa. Every habit you build, every skill you sharpen, every day you push through — it translates into XP, rank advancement, and titles.

You start as an **E-Rank Hunter**. The weakest. The forgettable one.

You end as the **Shadow Monarch**.

Or you don't. That part is up to you.

## DAILY CHALLENGES

Each day the System issues its orders. These are not optional.

Three categories of challenge exist, each targeting a different dimension of growth:

### 🌱 Growth
*"To make you grow."* — **XP Multiplier: 14.6×**

The hardest category. The highest reward. These are the challenges that push your intellect, your skills, your career. Reading. Learning. Building. Creating. Doing the difficult thing that your future self will thank you for.

### 🎮 Hobby
*"Things you like."* — **XP Multiplier: 7.3×**

The System recognizes that a Hunter who does nothing but grind will break. These are the pursuits that feed your soul — the things you do because they matter to you, not because they're useful.

### 💪 Self Care
*"Fitness. Diet. Recovery. Lifestyle."* — **XP Multiplier: 3.65×**

A weak body cannot carry a strong mind. These are the physical rituals — the workouts, the sleep discipline, the nutrition, the habits that build a vessel worthy of the power you seek.

### The Demands Scale With You

At E-Rank, the System asks for one challenge. As you rise, it asks for more. By the time you reach the peak, you face eight challenges daily across all three categories. The System does not ease up because you got stronger. It raises the bar.

### The Stakes

The higher your rank, the more XP hangs in the balance. Success is rewarded. Failure is punished.

| Rank | XP Gained (success) | XP Lost (failure) |
|------|---------------------|-------------------|
| E | 10 × multiplier | 0 |
| D | 20 × multiplier | 10 × multiplier |
| C | 30 × multiplier | 20 × multiplier |
| B | 40 × multiplier | 30 × multiplier |
| A | 50 × multiplier | 60 × multiplier |
| S | 60 × multiplier | 100 × multiplier |
| S2 | 70 × multiplier | 150 × multiplier |
| S3 | 80 × multiplier | 210 × multiplier |
| S4 | 90 × multiplier | 280 × multiplier |
| S5 | 100 × multiplier | 360 × multiplier |

At the lower ranks, failure carries no penalty — the System is patient with the weak. But cross into the elite tiers and the calculus flips: **failure costs more than success gains**. The System does not forgive the strong for being weak.

### Subtitles

Each daily challenge slot lets you record what you actually did — *"Ran 5km." "Finished the chapter." "Cooked instead of ordering."* Type it in before you mark the slot. It persists through the day, survives a page refresh, and disappears only when the day ends.

### Undo

The System is not a tyrant. Made a mistake? You can undo a marked challenge — reverting the XP and restoring the slot. But only for today. And only before midnight.

## QUESTS

Beyond the daily rhythm, the System occasionally issues a **Quest**.

Each day when you open the Quests page, the System rolls the dice. There is a small chance — a whisper of fate — that a quest appears for you that day. It is drawn at random from your personal **quest pool**: a collection of meaningful, ambitious challenges you've added yourself.

If you haven't built your quest pool, the System has nothing to offer you. **Build it.**

Quests are high-risk, high-reward. The System does not do things in halves:

| Outcome | Effect |
|---------|--------|
| **Complete** | +20× base XP gain + 1 Skip Pass |
| **Fail** | −30× base XP loss |

Choose wisely when the dice land in your favor.

---

## SKIP PASSES

A Skip Pass lets you mark one daily challenge slot as complete — without gaining or losing a single point of XP. No penalty. No shame. A free pass through a day you couldn't face.

You do not start with any. You earn them by completing Quests.

They are rare. They are precious. Use them on the days that truly demand it.

## HUNTER STATS

The System does not only measure XP. It watches your patterns across every day it has known you, and it calculates three core attributes:

**Intelligence** — How consistently do you pursue Growth? Every day the System has recorded you, how many Growth challenges did you actually clear? This is your mind, measured over time.

**Charisma** — How well do you take care of yourself? Your consistency in Self Care challenges, across your entire history in the System.

**Will Power** — The hardest one. How many of your total daily challenges — across all categories, across all days — have you actually completed? This is the truest measure of who you are.

These stats do not spike in a single heroic week. They are the slow accumulation of who you choose to be, every ordinary day.

## YOUR PROFILE

Your profile is your identity in the System:

- Your **rank banner** — a full-width image that changes as you advance through the ranks
- Your **rank emblem** — glowing in your rank's color, displayed prominently over the banner
- Your **title** — granted by the System at each rank milestone
- Your **XP progress bar** — showing exactly where you stand between your current rank and the next
- Your **hunter stats** — Intelligence, Charisma, Will Power

When you advance, everything changes. The image. The color. The title. The System makes sure you feel it.

## GETTING STARTED

### Prerequisites

- Node.js 18+
- npm 9+

### Setup

```bash
# Clone the repository
git clone <repo-url>
cd solo-leveling

# Install all dependencies
npm install

# Initialize the database
npm run db:migrate

# Seed with the starting player
npm run db:seed

# Start the System
npm run dev
```

The System awakens at:
- **Frontend:** http://localhost:3000
- **API:** http://localhost:3001

### Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start both client and server |
| `npm run dev:client` | Start frontend only |
| `npm run dev:server` | Start backend only |
| `npm run db:migrate` | Run database migrations |
| `npm run db:seed` | Seed the database with starting player |
| `npm run db:studio` | Open Prisma Studio (database GUI) |
